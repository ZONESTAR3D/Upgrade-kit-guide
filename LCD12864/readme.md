## LCD12864 (OLED12864) Menu Description
![](LCD12864.png)   ![](OLED12864.png)

### Function of knob
- **Clockwise rotation**: Menu item + / Value +
- **Counterclockwise rotation**: Menu item - / Value -
- **Push (click)**: Open Menu / Enter
